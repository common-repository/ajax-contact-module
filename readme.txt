== Description ==

Ajax Contact Module is a simple plugin that shows a contact module fixed at the bottom of the page in each of your blog pages.

For more information, check out <a href="http://www.danycode.com/ajax-contact-module/">Ajax Contact Module</a>

Plugin Features:

- Implemented with ajax
- After the form submission a cookie named acmcookie prevent to see again the form (delete this cookie to see again the module)
- Fully customizable

== Installation ==

1. Upload the Ajax Contact Module plugin to your blog and activate it. 
2. Customize the plugin through: Setting -> Ajax Contact Module
